# 使用帮助

## 快速开始

1. 打开"配置中心"，在订阅管理中添加 Azure Speech 的 Key 与 Region ✅
2. 选择输入来源（麦克风 / 回环）与设备 🎧
3. 点击"开始翻译"，等待字幕出现 ✨

## 输入源怎么选

- **麦克风**：适合会议、对话、现场收音
- **回环**：适合视频/系统声音，把"正在播放的声音"当输入
- **回环+麦克风**：同时采集系统声音与人声，适合需要双向收音的场景

## 字幕显示

- 需要时打开"浮动字幕"，方便边看边听
- 中间结果会实时刷新，最终结果会更稳定
- 浮动字幕支持调整透明度、字号与文字颜色

## 录音与历史

- 开启会话录音后，会先写 WAV，停止后自动转 MP3（Windows）
- MP3 码率等参数可在配置中心调整
- 历史记录会按会话累积，方便回看整理

## AI 复盘与洞察

- 在 Review 模式下，可对会话内容进行 AI 分析
- 内置预设洞察：会议总结、知识点提取、客诉识别、行动项提取、情绪分析
- 支持自定义 Prompt，也可开启自动洞察模式持续输出分析

## 批量字幕生成

- 在 Review 模式下，可对录音文件批量生成 SRT/VTT 字幕
- 需要配置 Azure Blob Storage 连接字符串（在配置中心设置）
- 支持字幕分句方式、最大字符数、时长等参数调整

## Media Studio

- 点击顶部"🎨 Media Studio"按钮进入
- 支持 AI 图片生成（DALL·E 等模型）和视频生成（Sora 2 等模型）
- 图片生成可挂载参考图（最多 8 张），内置裁剪工具
- 视频生成支持图生视频，可配置分辨率与时长
- 每个任务保存为独立会话，支持重命名、复制、删除

## 配置中心

- 点击顶部"配置中心"按钮打开
- **订阅管理**：添加/验证/测速多个 Azure Speech 订阅，支持中国区终结点
- **AI 配置**：设置 OpenAI 兼容 / Azure OpenAI 后端，支持 AAD 认证
- **录音设置**：开关录音、调整码率、自动删除 WAV 等
- **音频处理**：自动增益（AGC）、活动阈值等高级参数

## 常见问题

- **没有字幕**：确认订阅信息是否正确，网络是否正常
- **没声音/识别很少**：检查输入源与设备是否选对
- **回环无声音**：确认系统正在播放声音，并选择回环设备
- **AI 复盘无输出**：检查配置中心中 AI 后端的 Endpoint 与 Key 是否正确
- **批量字幕失败**：确认 Azure Blob Storage 连接字符串已正确配置
