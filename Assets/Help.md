# 使用帮助

## 快速开始

1. 在“订阅区域”填写 Azure Speech 的 Key 与 Region ✅
2. 选择输入来源（麦克风 / 回环）与设备 🎧
3. 点击“开始翻译”，等待字幕出现 ✨

## 输入源怎么选

- **麦克风**：适合会议、对话、现场收音
- **回环**：适合视频/系统声音，把“正在播放的声音”当输入

## 字幕显示

- 需要时打开“浮动字幕”，方便边看边听
- 中间结果会实时刷新，最终结果会更稳定

## 录音与历史

- 开启会话录音后，会先写 WAV，停止后自动转 MP3（Windows）
- 历史记录会按会话累积，方便回看整理

## 常见问题

- **没有字幕**：确认订阅信息是否正确，网络是否正常
- **没声音/识别很少**：检查输入源与设备是否选对
- **回环无声音**：确认系统正在播放声音，并选择回环设备
