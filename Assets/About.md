# 译见 Pro

译以载言，智以成文。

译见 Pro 把现场的声音转化为字幕，让信息即时可见；又用 AI 复盘把零散的片段整理为可追溯的结论、行动与风险。

## 功能概览

- **实时识别/翻译**：边播边出字幕（含中间结果/最终结果），支持语言自动检测。
- **输入源选择**：麦克风 / 回环（系统正在播放的声音），支持回环+麦克风混合模式。
- **浮动字幕**：可置顶显示，可调透明度与字号，适合边看视频/参加网络会议边看翻译。
- **会话录音**：翻译同时写 WAV，停止后自动转 MP3（Windows）。
- **AI 复盘与洞察**：对会话内容进行 AI 分析，提炼摘要、行动项、风险等，支持预设与自定义洞察。
- **批量字幕生成**：基于 Azure Speech Batch API 对录音文件批量生成 SRT/VTT 字幕。
- **Media Studio**：AI 图片与视频生成工作室，支持 DALL·E / Sora 2 等模型，支持参考图与图生视频。
- **多订阅管理**：通过配置中心管理多个 Azure Speech 订阅，支持验证与测速。
- **历史记录**：按会话积累文本，方便后续整理。
- **配置持久化**：本地保存，启动自动加载。

## 主要使用场景

- **会议纪要助手**：用麦克风收音，实时字幕 + 历史记录 + AI 复盘，方便会后整理。
- **边看边翻译在线视频**：回环作为输入 + 浮动字幕显示，让视频内容更易读。
- **录制/留存在线播放音频**：回环录制系统声音，得到 WAV/MP3 存档。
- **批量字幕与复盘**：对大量录音文件批量生成字幕并进行 AI 分析。
- **AI 内容创作**：通过 Media Studio 生成图片与视频素材。
